SOLUSI LENGKAP:

1. PASTIKAN STORAGE LINK AKTIF:
   php artisan storage:link

2. PASTIKAN FOLDER ADA:
   - storage/app/public/menus/ (untuk upload)
   - public/storage/menus/ (untuk akses web)

3. PERIKSA PERMISSION:
   - Pastikan folder bisa ditulis oleh web server

4. TEST UPLOAD MANUAL:
   - Login ke admin panel
   - Buka Menu → Tambah Menu
   - Upload gambar (PNG/JPG max 2MB)
   - Isi form lainnya
   - Save

5. PERIKSA DATABASE:
   - Gambar tersimpan dengan path: "menus/namafile.jpg"
   - Asset URL: http://localhost/storage/menus/namafile.jpg
